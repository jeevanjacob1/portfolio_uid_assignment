## Code Explanation
The portfolio was built from scratch utilizing standard web technologies to ensure a strong structural foundation without relying on heavy frameworks:
- **HTML5:** Used for the semantic structure of the website. Key sections include a sticky navigation bar, a hero section (Home), About Me, Projects, and Contact details. It is structured into logical `<section>` tags.
- **CSS3:** Applied a custom stylesheet (`style.css`) to implement a modern, dark-themed UI. 
  - **Flexbox & CSS Grid:** Utilized specifically in the projects section and navigation to ensure perfect alignment and responsiveness.
  - **Animations & Transitions:** Included smooth fade-ins and slide animations when the page loads, along with hover effects on buttons and project cards to make the user interface interactive.
  - **Responsiveness:** Employed media queries (`@media`) to ensure the website is fully functional on mobile devices, including transitioning the navbar to a hamburger menu.
